Copy this folder, rename it to your release slug, and replace the sample files.

Checklist:

1. Add a `cover.jpg` or `cover.png`
2. Put audio files inside `audio/`
3. Update `release.json`
4. Keep filenames ordered like `01-track-name.mp3`, `02-track-name.mp3`

If you remove the `tracks` array from `release.json`, the site will generate titles from the filenames automatically.
